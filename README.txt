Instructions rapides:
1. Place tous les fichiers sur un serveur web avec PHP (ex: /var/www/html/freshdrop).
2. Structure:
   - index.php
   - produit.php
   - header.php
   - footer.php
   - data/products.php
   - assets/css/style.css
   - assets/js/main.js
3. Ouvrir http://localhost/freshdrop/index.php
4. ajoute tes images (dans /assets/images ou URLs externes).
5. Pour production: ajouter validation côté serveur, gestion réelle de panier/commande (base de données), sécurité (sanitize, CSRF), paiement en ligne.
